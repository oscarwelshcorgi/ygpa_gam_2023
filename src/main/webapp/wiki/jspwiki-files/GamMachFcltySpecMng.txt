!!! 기계시설 제원 관리
[이미지]

!! 업무 내용
* 기계시설 제원을 등록 관리합니다.

!! 사용 설명  
* 조회
## 항구분, 시설물관리그룹, 시설분류, 시설명, 소재지를 입력하고 조회버튼을 클릭합니다.
* 입력
## 시설추가버튼을 클릭하여 기계시설 제원 화면에 해당하는 자료를 입력하고 저장버튼을클릭합니다.
* 수정
## 조회조건을 입력하여 수정할 자료를 조회합니다. 
## 기계시설 목록 화면에서 해당 자료를 선택하고 기계시설 제원을 클릭합니다.
## 기계시설 제원 화면에서 해당 자료를 수정하고 저장버튼을 클릭합니다.
* 삭제
## 조회조건을 입력하여 삭제할 자료를 조회합니다. 
## 기계시설 목록 탭 화면에서 삭제할 자료를 선택하고 삭제버튼을 클릭합니다.  
* 맵조회
## 조회조건을 입력하여 맵조회할 자료를 조회합니다. 
## 맵조회할 자료를 선택하고 맵조회버튼을 클릭합니다. 

!! 항목 설명
* 항구분: 관리기관입니다
* 시설물관리그룹: 시설물을 그룹으로 관리 합니다.
* 시설분류: 시설물을 관리기관으로 분류합니다.
* 시설명: 시설물 이름입니다.
* 소재지: 시설물 위치입니다.

!! 특이사항
* 기계시설 제원만 조회된다. (만약 조회결과에 해당 시설이 없을 경우 추가로 입력한다.)
* 맵조회는 그리드에서 해당 시설을 선택한 후 정상적으로 조회가 가능하다.

!!! 기계시설 제원 관리(기계시설 제원)
[이미지]

!! 업무 내용
* 기계시설 제원의 자료를 입력, 수정, 출력합니다.
* 기계시설 제원의 위치를 등록 조회 합니다.

!! 사용 설명
* 조회
## 기계시설 목록 화면에서 조회한 자료를 더블클릭합니다.
* 입력
## 기계시설 목록 화면에서 시설추가버튼을 클릭하여 기계시설 제원 화면에 해당하는 자료를입력하고 저장버튼을 클릭합니다.
* 수정
## 조회조건을 입력하여 수정할 자료를 조회합니다. 
## 기계시설 목록 화면에서 조회한 자료를 더블클릭합니다.
## 기계시설 제원 화면에서 해당 자료를 수정하고 저장버튼을 클릭합니다.
* 위치등록
## 위치등록버튼을 클릭하여 위치를 등록합니다.
* 위치조회
## 위치조회버튼을 클릭하여 위치를 조회합니다.

!! 항목 설명
* 항코드: 관리기관입니다
* GIS자산코드: 선택버튼을 클릭하여 GIS자산코드를 입력합니다.
* GIS자산명: GIS자산코드를 입력하면 자동 부여 됩니다.
* 지번: GIS자산코드를 입력하면 자동 부여 됩니다.
* 소재지: GIS자산코드를 입력하면 자동 부여 됩니다.
* 시설코드: 시설분류를 입력하면 자동 부여 됩니다.
* 시설분류: 시설물을 관리기관을 표시합니다.
* 시설명: 시설 이름을 표시합니다.
* 시설물관리그룹: 시설물을 그룹으로 관리 합니다.
* 설치일자: 시설물을 설치한 날짜입니다.
* 품명: 제품의 이름을 입력합니다
* 용도: 쓰이는 곳을 입력합니다.
* 형식: 시설의 형식을 입력합니다.
* 규격: 시설의 품질, 모양, 크기, 성능의 일정한 표준을 입력합니다.
* 길이: 시설의 길이를 입력합니다.
* 수량: 시설의 개수를 입력합니다.
* 단위: 시설의 단위를 입력합니다.
* 흘수: 시설의 흘수를 입력합니다.
* 장비번호: 장비의 번호를 입력합니다.
* 상태등급: 상태를 나타내는 등급을 입력합니다.
* 제작회사: 제품을 만든 회사이름을 입력합니다.
* 제작일자: 제품의 제작 날짜를 입력합니다.
* 하역능력: 시설의 하역능력을 입력합니다.
* 내용년수: 고정자산이 경제적으로 사용될 수 있는 이용기간 입력합니다.
* 자중: 중력에 의해 생기는 힘을 입력합니다.
* 레일간격: 레일의 간격을 입력합니다.
* 정격하중: 사용조건과 그 성능의 범위의 무게를 입력합니다.
* 최대바퀴하중: 바퀴의 성능범위의 무게를 입력합니다.
* 적재톤수: 물건이나 짐을 실을 수 있는 톤수를 입력합니다.
* 강제중량: 최대 적재 중량을 입력합니다.
* 정격마력: 사용조건과 그 성능의 범위의 힘을 입력합니다.
* 처리능력: 시설이 처리할 수 있는 능력을 입력합니다.
* 사용업체: 시설을 사용하는 업체 이름을 입력합니다.
* 운영회사: 시설을 운영하는 회사의 이름을 입력합니다.
* 사용목적: 시설을 사용할 목적을 입력합니다.
* 관리자: 시설을 관리하는 사람을 입력합니다.
* 사용시작일자: 시설을 사용 시작하는 날짜를 입력합니다.
* 사용종료일자: 시설을 사용 종료하는 날짜를 입력합니다.
* 소유주체: 시설을 소유한 기관을 입력합니다.
* 취득금액: 시설을 취득한 액수를 입력합니다.
* 검사기관: 시설을 검사한 기관 이름을 입력합니다.
* 검사합격번호: 시설을 검사합격 한 번호를 입력합니다.
* 검사시작일자: 시설을 검사 시작한 날짜를 입력합니다.
* 검사종료일자: 시설을 검사 종료한 날짜를 입력합니다.
* 시설물분류코드: 선택버튼을 클릭하여 분류코드를 입력합니다.
* 건축시설물관리번호: 선택버튼을 클릭하여 관리번호를 입력합니다.
* 소재지: 시설의 위치를 입력합니다.
* 비고: 기타 사항을 입력합니다.

!! 특이사항
* 기계시설 일반에서는 GIS 자산 및 시설분류, 시설물 관리 그룹은 반드시 입력해야 한다.
* 기계시설 제원에서는 필요한 항목만 입력해도 무방하다.

!!! 기계시설 제원 관리(기계시설 첨부파일)
[이미지]

!! 업무 내용
* 기계시설 제원의 문서, 사진을 업로드, 다운로드, 삭제, 저장합니다.
* 업로드 한 사진을 미리보기 합니다.

!! 사용 설명
* 조회
## 기계시설 목록 화면에서 조회한 자료를 선택하고 기계시설 첨부파일 탭을 클릭합니다.  
* 업로드
## 업로드버튼을 클릭합니다.
## 첨부파일 업로드화면에서 파일추가버튼을 클릭합니다.
## 업로드 할 파일을 선택하고 확인버튼을 클릭합니다.
## 기계시설 첨부파일 화면에 저장버튼을 클릭합니다.
* 다운로드
## 다운로드할 자료를 선택하고 다운로드버튼을 클릭합니다.
* 삭제
## 삭제할 자료를 선택하고 삭제버튼을 클릭합니다.

!! 항목 설명
* 입력항목 설명
## 파일구분: 문서인지 사진인지 입력합니다.
## 파일제목: 파일명이 아닌 파일의 이름을 입력합니다.

!! 특이사항
*첨부파일은 문서, 사진, 기타로 구분된다. (사진일 경우 우측에서 프리뷰 가능)
