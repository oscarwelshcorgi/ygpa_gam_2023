!!! 시설물 하자보수 관리
[이미지]

!! 업무 내용
* 시설물관리자가 시설물 하자보수 내역을 조회, 삭제, 추가, 엑셀 다운로드, 
  하자검사결과 인쇄 합니다.

!! 사용 설명  
* 조회
## 시설물업무구분, 계약명, 하자보수구분, 하자검사기간을 입력하여 조회버튼을 클릭합니다.
* 하자검사결과인쇄
## 조회조건을 입력하여 인쇄할 자료를 조회합니다. 
## 하자검사결과인쇄버튼을 클릭하여, 조회된 리스트 인쇄양식창을 띄웁니다.
## 인쇄양식창에서 인쇄버튼을 클릭하여 출력합니다.
* 엑셀
## 조회조건을 입력하여 다운로드할 자료를 조회합니다. 
## 엑셀버튼을 클릭하여, 조회된 리스트를 엑셀로 다운로드 합니다. 
* 추가
## 신규 하자보수내역을 입력하고자 할 때 추가 버튼을 클릭합니다. 
## 버튼 클릭 시 하자보수내역 상세 Tab로 넘어가면서 입력상태가 됩니다.  
* 삭제
## 조회조건을 입력하여 삭제할 자료를 조회합니다. 
## 하자보수 리스트화면에서 삭제할 자료를 선택하고 삭제버튼을 클릭합니다.  
  
!! 항목 설명
* 조회항목 설명
## 시설물업무구분: 시설물구분값을 선택합니다.
  => 전기시설물, 기계시설물, 토목시설물, 건축시설물, 정보통신시설물
## 계약명: 하자보수 입력/수정 시 선택한 계약명입니다.
## 하자검사구분: 하자검사구분을 선택합니다.
  => 상반기, 하반기
## 하자검사기간: 하자보수 입력/수정 시 입력한 하자검사일자입니다. 
  => 입력한 날짜 범위내의 모든 하자보수 내역을 조회합니다.

!! 특이사항
* 업무 구분을 정확히 확인한 후 사용한다.
(건축시설, 토목시설, 기계시설, 전기시설, 정보통신시설)
* 시설물 관리 그룹이 없을 경우 시설물 관리 그룹 관리 화면에서 확인한다.
* 계약정보가 없을 경우 시설물 계약정보 관리 화면에서 확인한다.


!!! 시설물 하자보수 관리(하자보수이력상세)
[이미지]

!! 업무 내용
* 하자보수내역목록을 선택했을 시 해당 하자보수 상세내역을 출력합니다. 
* 하자보수내역목록을 선택했을 시 하자보수 내역을 수정합니다.
* 하자보수내역목록에서 추가버튼을 클릭했을 시 하자보수 내역을 입력합니다.

!! 사용 설명  
* 조회
## 시설물업무구분, 계약명, 하자보수구분, 하자검사기간을 입력하여 조회버튼을 클릭합니다.
## 조회버튼 클릭 시 하자보수내역 목록으로 Tab이 이동되며 조회조건에 해당하는 
  목록을 출력합니다.
* 하자검사조서인쇄
## 하자검사조서인쇄버튼을 클릭하여, 인쇄양식창을 띄웁니다.
## 인쇄양식창에서 인쇄버튼을 클릭하여 출력합니다.
* 하자만료검사조서인쇄
## 하자만료검사조서인쇄버튼을 클릭하여, 인쇄양식창을 띄웁니다.
## 인쇄양식창에서 인쇄버튼을 클릭하여 출력합니다.
* 저장
## 하자보수 내역을 입력/수정 처리합니다.
## 하자보수내역목록을 선택했을 시 수정 처리합니다. 
## 하자보수내역목록에서 추가버튼을 클릭했을 시 입력 처리합니다.
## 저장은 하자보수내역상세, 하자보수대상시설물, 하자검사자, 하자보수첨부파일Tab에서 
  처리된 내용 모두를 일괄 저장합니다.

!! 항목 설명* 시설물관리그룹: 입력 시 유지보수 대상 시설물의 관리그룹을 선택합니다. 
  수정 시는 선택할 수 없습니다.
* 시행년도: 유지보수 시행년도를 선택합니다.
* 시설물업무구분: 입력 시 시설물 구분값을 선택합니다. 수정 시는 선택되지 않습니다.
  => 전기시설물, 기계시설물, 토목시설물, 건축시설물, 정보통신시설물
* 계약번호: 해당시설물의 계약번호를 선택합니다.
* 도급엽체명: 계약번호 선택 시 자동 입력됩니다. 
* 순번: 입력되는 순서대로 자동입력 처리되며, 수정되지 않습니다.
* 하자검사구분: 하자검사구분을 선택합니다.
  => 상반기, 하반기
* 하자검사일자: 하자검사일자를 입력합니다.
* 하자유무: 하자유무를 선택합니다.
* 하자발생일자: 하자발생일자를 입력합니다.
* 하자보수기간: 하자보수기간을 입력합니다.
* 하자보수유형: 하자보수유형을 선택합니다.
* 하자보수금액: 하자보수금액을 입력합니다.
* 하자보수완료여부: 하자보수완료여부를 선택합니다.
* 하자보수내용: 실제 하자보수된 상세 내용을 입력합니다.
* 비고: 기타 사항을 입력합니다.

!! 특이사항
* 업무 구분 및 시설물 관리 그룹은 반드시 입력해야 한다.
* 시설물 관리 그룹이 없을 경우 시설물 관리 그룹 관리 화면에서 확인한다.
* 계약정보가 없을 경우 시설물 계약정보 관리 화면에서 확인한다.


!!! 시설물 하자보수 관리(하자보수대상시설물)
[이미지]

!! 업무 내용
* 하자보수 대상시설물 목록을 출력합니다. 
* 하자보수 대상시설물 내역을 수정/삭제합니다.

!! 사용 설명  
* 조회
## 시설물업무구분, 계약명, 하자보수구분, 하자검사기간을 입력하여 조회버튼을 클릭합니다.
## 조회버튼 클릭 시 하자보수내역 목록으로 Tab이 이동되며 조회조건에 해당하는 
  목록을 출력합니다.
* 추가
## 하자보수대상시설물을 추가합니다.
## 추가버튼을 누르면 목록 하위에 빈 목록이 생성이 되며, 아래의 입력창에 입력을 하면
  목록에 반영이 됩니다.
## 저장 버튼을 누르기 전에는 화면상에만 변경된 것이며, 저장된 상태가 아닙니다.
* 삭제
## 하자보수대상시설물을 삭제합니다.
## 목록을 선택하고 삭제버튼을 누르면 해당하는 목록이 사라집니다.
## 저장 버튼을 누르기 전에는 화면상에만 변경된 것이며, 저장된 상태가 아닙니다.
* 목록선택
## 목록을 선택하면 선택한 목록의 내용이 하단 입력부에 표시되면 수정상태가 됩니다.
## 하단입력부에 표시된 내용을 수정하면 목록에 바로 반영이 됩니다.
## 저장 버튼을 누르기 전에는 화면상에만 변경된 것이며, 저장된 상태가 아닙니다.
* 저장
## 하자보수 내역을 입력/수정 처리합니다.
## 하자보수내역목록을 선택했을 시 수정 처리합니다. 
## 하자보수내역목록에서 추가버튼을 클릭했을 시 입력 처리합니다.
## 저장은 하자보수내역상세, 하자보수대상시설물, 하자검사자, 하자보수첨부파일Tab에서
  처리된 내용 모두를 일괄 저장합니다. 

!! 항목 설명
* 대상시설물: 하자보수내역상세 Tab의 시설물업무구분에 해당하는 대상시설물을
 선택합니다
* 하자검사일자: 하자검사일자를 입력합니다.
* 하자유무: 하자유무를 선택합니다.
* 하자검사결과: 하자검사결과를 입력합니다.
* 비고: 하자보수에 대한 기타사항을 입력합니다.

!! 특이사항
* 대상시설물은 반드시 입력해야 한다.
* 대상시설물이 없을 경우 각 시설물 제원 관리 화면에서 확인한다.
