!!! 시설물 점검이력 조회
[이미지]

!! 업무 내용
* 점검 시설물에 대한 이력을 조회합니다.

!! 사용 설명  
* 조회
## 점검진단기관명, 점검시설명, 시설점검기간을 입력하고 조회버튼을 누릅니다.

!! 항목 설명
* 조회항목 설명
## 점검진단기관명: 조회할 점검진단기관명을 입력합니다.
## 점검시설명: 조회할 시설명을 선택하여 입력합니다.
## 시설점검기간: 조회할 시설점검기간을 달력으로 선택하여 입력합니다.

!! 특이사항
* 업무 구분을 정확히 확인한 후 사용한다.(건축시설, 토목시설, 기계시설, 전기시설, 정보통신시설)
* 시설물 관리 그룹이 없을 경우 시설물 관리 그룹 관리 화면에서 확인한다.


!!! 시설물 점검이력 조회(점검이력내역)
[이미지]

!! 업무 내용
* 점검이력목록에서 선택한 점검이력내역을 조회합니다.

!! 사용 설명  
* 조회
## 점검이력목록에서 내역을 선택한 후 점검이력내역 탭으로 이동합니다.

!! 항목 설명
* 시설물: 점검이력 대상이 되는 시설물을 조회합니다.
* 점검자: 해당 시설물의 점검자를 조회합니다.
* 점검일자: 해당 시설물에 대한 점검진단을 시행한 날짜를 조회합니다.
* 상태평가등급: 해당 시설물의 상태평가등급을 조회합니다.
* 점검진단결과: 해당 시설물에 대한 점검진단을 시행한 결과를 조회합니다.
* 비고: 해당 시설물에 대한 비고를 조회합니다.
* 관리그룹: 해당 시설물이 포함되어 있는 점검관리내역의 관리그룹명을 조회합니다.
* 점검관리순번: 해당 시설물이 포함되어 있는 점검관리내역의 점검관리순번을 조회합니다.
* 업무구분: 해당 시설물이 포함되어 있는 점검관리내역의 업무구분을 조회합니다.
* 점검관리명: 해당 시설물이 포함되어 있는 점검관리내역의 점검관리명을 조회합니다.
* 시행년도: 해당 시설물이 포함되어 있는 점검관리내역의 시행년도를 조회합니다.
* 점검구분: 해당 시설물이 포함되어 있는 점검관리내역의 점검구분을 조회합니다.
* 시행일자: 해당 시설물이 포함되어 있는 점검관리내역의 시행일자를 조회합니다.
* 점검진단자: 해당 시설물이 포함되어 있는 점검관리내역의 점검진단자를 조회합니다.
* 점검진단구분: 해당 시설물이 포함되어 있는 점검관리내역의 점검진단구분을 조회합니다.
* 점검진단금액: 해당 시설물이 포함되어 있는 점검관리내역의 점검진단비용을 조회합니다.
* 상태평가등급: 해당 시설물이 포함되어 있는 점검관리내역의 상태평가등급을 조회합니다.
* 점검진단기관명: 해당 시설물이 포함되어 있는 점검관리내역의 점검진단기관명을 조회합니다.
* 점검기간: 해당 시설물이 포함되어 있는 점검관리내역의 점검시작일자와 점검종료일자를 조회합니다.
* 책임기술자명: 해당 시설물이 포함되어 있는 점검관리내역의 책임기술자명을 조회합니다.
* 점검진단결과: 해당 시설물이 포함되어 있는 점검관리내역의 점검진단결과를 조회합니다.
* 조치내용: 해당 시설물이 포함되어 있는 점검관리내역의 조치내용을 조회합니다.
* 비고: 해당 시설물이 포함되어 있는 점검관리내역의 비고를 조회합니다.
