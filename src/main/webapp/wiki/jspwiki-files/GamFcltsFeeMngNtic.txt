!!시설물 관리비 고지

!1. 업무 내용
*시설물의 관리비를 고지처리, 고지취소, 고지서출력, 추가고지 입력, 추가고지 삭제를 합니다.

!2. 사용 설명
*조회 - 관리기간, 업무구분, 고지기간, 수납구분, 요금 종류 등을 선택 후 조회버튼을 클릭합니다.
*고지처리 - 시설물관리비 고지현황 내역에서 고지할 자료를 선택합니다.\\　　　　 - 고지처리 버튼을 클릭하여 고지를 실행합니다.
*고지 취소 - 시설물관리비 고지현황 내역에서 고지 취소할 자료를 선택합니다.\\　　　 　 - 고지취소 버튼을 클릭하면 고지금액과 함께 경고창이 뜨며 확인을 눌러 고지를 취소합니다.
*고지서 출력 - 시설물관리비 고지현황 내역에서 고지서 출력할 자료를 선택합니다.\\　　　 　　 - 고지서 출력버튼을 클릭하여 고지서를 출력합니다.
*추가고지 입력 - 추가고지 할 자료를 선택 후 추가고지 입력 버튼을 클릭하여 시설물 관리비 고지내역 화면으로\\　　　　 　　　 이동합니다.\\　　　　　　　- 시설물 관리비 고지내역 화면에서 다시 한 번 추가고지 입력 버튼을 클릭합니다.\\　　　　　　　- 입력항목에 추가고지 할 내역을 입력하고 저장 버튼을 클릭합니다.
*추가고지 취소 - 추가고지를 취소할 자료를 선택 후 초가고지 취소 버튼을 클릭합니다.
*엑셀 다운로드 - 리스트화면의 내용을 엑셀파일로 다운로드합니다. 
*납부현황 조회 - 납부현황조회를 할 자료를 선택한 후 납부현황 조회 버튼을 클릭합니다.

!3. 예외 설명
*노란색으로 표기된 항목은 고지금액이 연체되어 있는 상태입니다.
*고지처리가 되어있지 않을 경우에는 취소, 출력 등 일부 버튼은 활성화 되지 않습니다.
*조회조건에 맞는 자료가 없을 경우 “해당 조건의 자료가 존재하지 않습니다.”라는 메시지가 출력됩니다.
