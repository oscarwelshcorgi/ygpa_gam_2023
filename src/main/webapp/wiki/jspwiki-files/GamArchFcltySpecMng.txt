!!! 건축시설 제원 관리
[이미지]

!! 업무 내용
* 건축시설 제원을 등록 관리합니다.

!! 사용 설명  
* 조회
## 항구분, 시설물관리그룹, 시설분류, 시설명, 소재지를 입력하고 조회버튼을 클릭합니다.
* 입력
## 시설추가버튼을 클릭하여 건축시설 제원 화면에 해당하는 자료를 입력하고 저장버튼을 클릭합니다.
* 수정
## 조회조건을 입력하여 수정할 자료를 조회합니다. 
## 건축시설 목록 화면에서 해당 자료를 더블클릭합니다.
## 건축시설 제원 화면에서 해당 자료를 수정하고 저장버튼을 클릭합니다.
* 추가
## 추가버튼을 클릭합니다.
## 건축시설 제원 화면에 해당하는 자료를 입력하고 저장버튼을 클릭합니다.
* 삭제
## 조회조건을 입력하여 삭제할 자료를 조회합니다. 
## 건축시설 목록 화면에서 삭제할 자료를 선택하고 삭제버튼을 클릭합니다.  
* 엑셀 다운로드
## 조회조건을 입력하여 맵조회할 자료를 조회합니다.
## 조회된 내용을 엑셀파일로 저장합니다.
* 맵조회
## 조회조건을 입력하여 맵조회할 자료를 조회합니다. 
## 맵조회할 자료를 선택하고 맵조회버튼을 클릭합니다.
## 건축시설 제원의 위치가 지도에 표시됩니다.
* 맵편집
## 맵편집 버튼을 클립합니다.
## 편집하고자 하는 기능을 선택하여 사용합니다.(점, 선을 사용하여 지도 데이터 입력 또는 기존 선 수정가능)

!! 항목 설명
* 항구분: 관리기관입니다
* 시설물관리그룹: 시설물을 그룹으로 관리 합니다.
* 시설분류: 시설물을 관리기관으로 분류합니다.
* 시설명: 시설물 이름입니다.
* 소재지: 시설물 위치입니다.

!!! 건축시설 제원 관리(건축시설 제원)
[이미지]

!! 업무 내용
* 건축시설 제원의 자료를 수정, 출력합니다.
* 건축시설 제원의 위치를 등록 조회 합니다.

!! 사용 설명
* 조회
## 건축시설 목록 화면에서 조회한 자료를 더블클릭합니다.
* 처음으로
## 조회된 데이터의 처음으로 이동합니다.
* 이전자료
## 조회된 데이터의 이전 자료로 이동합니다.
* 다음자료
## 조회된 데이터의 다음 자료로 이동합니다.
* 마지막자료
## 조회된 데이터의 마지막 자료로 이동합니다.
* 입력
## 건축시설 목록 화면에서 시설추가버튼을 클릭하여 건축시설 제원 화면에 해당하는 자료를 입력하고 저장버튼을 클릭합니다.
* 저장
## 수정 및 추가 할 데이터를 입력하고 저장을 하면 최종데이터가 저장 됩니다.
* 삭제
## 화면에 출력된 내용을 삭제 합니다.(삭제된 내용을 박원이 불가능 합니다.)
* 복사추가
## 화면의 내용을 복사하여 새로운 데이터를 생성합니다.
## 복사 후 수정할 내용을 변경 후 저장 버튼을 클릭
## 새로생성된 데이터를 저장합니다.

!! 항목 설명
* 항코드: 관리기관입니다
* GIS자산코드: 선택버튼을 클릭하여 GIS자산코드를 입력합니다.
* GIS자산명: GIS자산코드를 입력하면 자동 부여 됩니다.
* 지번: GIS자산코드를 입력하면 자동 부여 됩니다.
* 소재지: GIS자산코드를 입력하면 자동 부여 됩니다.
* 시설코드: 시설분류를 입력하면 자동 부여 됩니다.
* 시설분류: 시설물을 관리기관을 표시합니다.
* 건축시설명: 건축시설 이름을 표시합니다.
* 시설물관리그룹: 시설물을 그룹으로 관리 합니다.
* 준공일자: 공사를 다 마친 날짜입니다.
* 구조형식: 건물의 주요 구조부를 구성하는 형식입니다.
* 하자만료일자: 하자를 끝내는 날입니다.
* 기초형식: 건물의 기초를 구성하는 형식입니다.
* 대지면적: 땅의 넓이입니다.
* 건축면적: 건물의 바닥면적입니다.
* 연면적: 하나의 건축물의 각 층 바닥면적의 합계입니다.
* 주사용용도: 주로 사용할 목적을 입력합니다.
* 주차면적: 주차 넓이입니다.
* 주차대수: 주차가능 차 대수입니다.
* 옥외주차면적: 옥외 주차 넓이입니다.
* 옥외주차대수: 옥외 주차가능 차 대수입니다.
* 옥내주차면적: 옥내 주차 넓이입니다.
* 옥내주차대수: 옥내 주차가능 차 대수입니다.
* 배기닥트유무: 환풍기의 유무입니다.
* 환기공조방식: 환기 방식을 입력합니다.
* 승강기운영방식: 승강기 운영 방식을 입력합니다.
* 승강기대수승객용: 승객용 승강기 대수입니다.
* 승강기대수비상용: 비상용 승강기 대수입니다.
* 승강기대수화물용: 화물용 승강기 대수입니다.
* 전기인입용량: 전기가 들어오는 용량을 입력합니다.
* 변전실위치: 변전실의 위치를 입력합니다.
* 정화조형식: 정화조의 구조를 입력합니다.
* 오수정화시설위치: 오수정화시설의 위치를 입력합니다.
* 물탱크위치: 물탱크의 위치를 입력합니다.
* 유류저장시설위치: 유류저장시설의 위치를 입력합니다.
* 냉방유무: 냉방의 유무를 입력합니다.
* 냉방열원: 냉방의 공급원을 입력합니다.
* 난방유무: 난방의 유무를 입력합니다.
* 난방열원: 난방의 공급원을 입력합니다.
* 외장: 건물의 외부에 쓰는 자재를 입력합니다.
* 내장: 건물의 내부에 쓰는 자재를 입력합니다.
* 천장: 건물의 천장에 쓰는 자재를 입력합니다.
* 지붕: 건물의 지붕에 쓰는 자재를 입력합니다.
* 지붕방수: 건물의 지붕방수에 쓰는 자재를 입력합니다.
* 설계공사명: 설계공사의 이름입니다.
* 설계시작일: 설계를 시작한 날짜입니다.
* 설계종류일자: 설계를 다 마친 날짜입니다.
* 시공공사명: 공사를 실시하는 공사의 이름을 입력합니다.
* 시공수행회사: 공사를 실시하는 수행회사의 이릅을 입력합니다.
* 시공시작일: 공사를 시작하는 날짜입니다.
* 시공종료일자: 공사를 다 마치는 날짜입니다.
* 건축시설물분류코드: 선택버튼을 클릭하여 분류코드를 입력합니다.
* 소재지: 시설물 위치를 입력합니다.
* 비고: 기타 사항을 입력합니다.

!! 특이사항
* 건축시설 일반에서는 GIS 자산 및 시설분류, 시설물 관리 그룹은 반드시 입력해야 한다.
* 건축시설 제원에서는 필요한 항목만 입력해도 무방하다.

!!! 건축시설 제원 관리(건축시설 첨부파일)
[이미지]

!! 업무 내용
* 건축시설 제원의 문서, 사진을 업로드, 다운로드, 삭제, 저장합니다.
* 사진을 미리보기 합니다.

!! 사용 설명
* 조회
## 건축시설 목록 화면에서 조회한 자료를 선택하고 건축시설 첨부파일 탭을 클릭합니다.
* 업로드
## 업로드버튼을 클릭합니다.
## 첨부파일 업로드화면에서 파일추가버튼을 클릭합니다.
## 업로드 할 파일을 선택하고 확인버튼을 클릭합니다.
## 건축시설 첨부파일 화면에 저장버튼을 클릭합니다.
* 다운로드
## 다운로드할 자료를 선택하고 다운로드버튼을 클릭합니다.
* 삭제
## 삭제할 자료를 선택하고 삭제버튼을 클릭합니다.
!! 항목 설명
* 입력항목 설명
## 파일구분: 문서인지 사진인지 입력합니다.
## 파일제목: 파일명이 아닌 파일의 이름을 입력합니다.

!! 특이사항 
* 첨부파일은 문서, 사진, 기타로 구분된다. (사진일 경우 우측에서 프리뷰 가능)
