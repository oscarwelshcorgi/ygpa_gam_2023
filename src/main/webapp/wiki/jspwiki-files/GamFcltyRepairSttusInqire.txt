!!! 시설물 하자보수현황 조회
[이미지]

!! 업무 내용
* 시설물관리자가 시설물 하자보수 내역을 조회합니다.

!! 사용 설명  
* 조회
## 시설물업무구분, 계약명, 하자보수구분, 하자검사기간을 입력하여 조회버튼을 클릭합니다.
## 조회버튼 클릭 시 하자보수내역 목록으로 Tab이 이동되며 조회조건에 해당하는 
  목록을 출력합니다.

!! 항목 설명
* 시설물업무구분: 시설물구분값을 선택합니다.
  => 전기시설물, 기계시설물, 토목시설물, 건축시설물, 정보통신시설물
* 계약명: 하자보수 입력/수정 시 선택한 계약명입니다.
* 하자검사구분: 하자검사구분을 선택합니다.
  => 상반기, 하반기
* 하자검사기간: 하자보수 입력/수정 시 입력한 하자검사일자입니다. 
  => 입력한 날짜 범위내의 모든 하자보수 내역을 조회합니다.
!! 특이사항
* 업무 구분을 정확히 확인한 후 사용한다.
(건축시설, 토목시설, 기계시설, 전기시설, 정보통신시설)
* 시설물 관리 그룹이 없을 경우 시설물 관리 그룹 관리 화면에서 확인한다.
* 계약정보가 없을 경우 시설물 계약정보 관리 화면에서 확인한다.

!!! 시설물 하자보수현황 조회(하자보수이력상세)
[이미지]

!! 업무 내용
* 하자보수내역목록을 선택했을 시 해당 하자보수 상세내역을 출력합니다. 

!! 사용 설명  
* 조회
## 시설물업무구분, 계약명, 하자보수구분, 하자검사기간을 입력하여 조회버튼을 클릭합니다.
## 조회버튼 클릭 시 하자보수내역 목록으로 Tab이 이동되며 조회조건에 해당하는 
  목록을 출력합니다.

!! 항목 설명
* 시설물관리그룹: 유지보수 대상 시설물의 관리그룹을 출력합니다.  
* 시행년도: 유지보수 시행년도를 출력합니다.
* 시설물업무구분: 입력 시 시설물 구분값을 출력합니다.
  => 전기시설물, 기계시설물, 토목시설물, 건축시설물, 정보통신시설물
* 계약번호: 해당시설물의 계약번호를 출력합니다.
* 도급엽체명: 도급업체명을 출력합니다. 
* 순번: 순번을 출력합니다.
* 하자검사구분: 하자검사구분을 출력합니다.
  => 상반기, 하반기
* 하자검사일자: 하자검사일자를 출력합니다.
* 하자유무: 하자유무를 출력합니다.
* 하자발생일자: 하자발생일자를 출력합니다.
* 하자보수기간: 하자보수기간을 출력합니다.
* 하자보수유형: 하자보수유형을 출력합니다.
* 하자보수금액: 하자보수금액을 출력합니다.
* 하자보수완료여부: 하자보수완료여부를 출력합니다.
* 하자보수내용: 하자보수된 상세 내용을 출력합니다.
* 비고: 기타 사항을 출력합니다.

!!! 시설물 하자보수현황 조회(하자보수대상시설물)
[이미지]

!! 업무 내용
* 하자보수 대상시설물 목록을 출력합니다. 

!! 사용 설명  
* 조회
## 시설물업무구분, 계약명, 하자보수구분, 하자검사기간을 입력하여 조회버튼을 클릭합니다.
## 조회버튼 클릭 시 하자보수내역 목록으로 Tab이 이동되며 조회조건에 해당하는 
  목록을 출력합니다.

!! 항목 설명
* 대상시설물: 하자보수내역상세 Tab의 시설물업무구분에 해당하는 대상시설물을
 출력합니다
* 하자검사일자: 하자검사일자를 출력합니다.
* 하자유무: 하자유무를 출력합니다.
* 하자검사결과: 하자검사결과를 출력합니다.
* 비고: 하자보수에 대한 기타사항을 출력합니다.
