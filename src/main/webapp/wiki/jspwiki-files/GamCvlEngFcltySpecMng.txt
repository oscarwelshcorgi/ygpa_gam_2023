!!! 토목시설 제원 관리
[이미지]

!! 업무 내용
* 토목시설 제원을 등록 관리합니다.

!! 사용 설명  
* 조회
## 항구분, 시설물관리그룹, 시설분류, 시설명, 소재지를 입력하고 조회버튼을 클릭합니다.
* 입력
## 시설추가버튼을 클릭하여 토목시설 제원 화면에 해당하는 자료를 입력하고 저장버튼을클릭합니다.
* 수정
## 조회조건을 입력하여 수정할 자료를 조회합니다. 
## 토목시설 목록 화면에서 해당 자료를 더블클릭합니다.
## 토목시설 제원 화면에서 해당 자료를 수정하고 저장버튼을 클릭합니다.
* 삭제
## 조회조건을 입력하여 삭제할 자료를 조회합니다. 
## 토목설 목록 화면에서 삭제할 자료를 선택하고 삭제버튼을 클릭합니다.  
* 맵조회
## 조회조건을 입력하여 맵조회할 자료를 조회합니다. 
## 맵조회할 자료를 선택하고 맵조회버튼을 클릭합니다. 

!! 항목 설명
* 항구분: 관리기관입니다
* 시설물관리그룹: 시설물을 그룹으로 관리 합니다.
* 시설분류: 시설물을 관리기관으로 분류합니다.
* 시설명: 시설물 이름입니다.
* 소재지: 시설물 위치입니다.

!! 특이사항
* 토목시설 제원만 조회된다. (만약 조회결과에 해당 시설이 없을 경우 추가로 입력한다.)
* 맵조회는 그리드에서 해당 시설을 선택한 후 정상적으로 조회가 가능하다.


!!! 토목시설 제원 관리(토목시설 제원)
[이미지]

!! 업무 내용
* 토목시설 제원의 자료를 입력, 수정, 출력합니다.
* 토목시설 제원의 위치를 등록 조회 합니다.

!! 사용 설명
* 조회
## 토목시설 목록 화면에서 조회한 자료를 더블클릭합니다.
* 입력
## 토목시설 목록 화면에서 시설추가버튼을 클릭하여 토목시설 제원 화면에 해당하는 자료를 입력하고 저장버튼을 클릭합니다.
* 수정
## 조회조건을 입력하여 수정할 자료를 조회합니다. 
## 토목시설 목록 화면에서 조회한 자료를 더블클릭합니다.
## 토목시설 제원 화면에서 해당 자료를 수정하고 저장버튼을 클릭합니다.
* 위치등록
## 위치등록버튼을 클릭하여 위치를 등록합니다.
* 위치조회
## 위치조회버튼을 클릭하여 위치를 조회합니다.

!! 항목 설명
* 입력항목 설명
* 항코드: 관리기관입니다
* GIS자산코드: 선택버튼을 클릭하여 GIS자산코드를 입력합니다.
* GIS자산명: GIS자산코드를 입력하면 자동 부여 됩니다.
* 지번: GIS자산코드를 입력하면 자동 부여 됩니다.
* 소재지: GIS자산코드를 입력하면 자동 부여 됩니다.
* 시설코드: 시설분류를 입력하면 자동 부여 됩니다.
* 시설분류: 시설물을 관리기관을 표시합니다.
* 토목시설명: 건축시설 이름을 표시합니다.
* 시설물관리그룹: 시설물을 그룹으로 관리 합니다.
* 설치일자: 시설물을 설치한 날짜입니다.
* 변경일자: 시설물을 변경한 날짜입니다.
* 시설물연장: 시설물의 연장한 수치를 입력합니다.
* 천단표고: 천단의 표고를 입력합니다.
* 천단폭: 천단의 폭을 입력합니다.
* 선좌수심: 선좌의 바다의 깊이를 입력합니다.
* 구조형식: 건물의 주요 구조부를 구성하는 형식을 입력합니다.
* 상치폭: 상치의 폭을 입력합니다.
* 에이프런폭: 에이프런의 폭을 입력합니다.
* 에이프런포장종류: 에이프런의 포장종류를 입력합니다.
* 에이프런포장구배: 에이프런의 포장구배를 입력합니다.
* 접안선박규모: 접안한 선박의 크기를 입력합니다.
* 상재하중: 상재의 무게를 입력합니다.
* 기초저면토질: 기초밑면의 토질을 입력합니다.
* 취급화물: 취급하는 화물을 입력합니다.
* 말뚝구경: 말뚝의 지름을 입력합니다.
* 말뚝연장: 말뚝의 연장을 입력합니다.
* 말뚝본수: 말뚝의 개수를 입력합니다.
* 급수전수량: 급수전 수량을 입력합니다.
* 널말뚝규격: 널말뚝의 규격을 입력합니다.
* 소화전수량: 소화전의 수량을 입력합니다.
* 선석: 접안장소를 입력합니다.
* 야적장포장종류: 야적장의 포장 종류를 입력합니다.
* 야적장면적: 야적장의 넓이를 입력합니다.
* 방충재종류코드: 방충재의 종류코드를 입력합니다.
* 방충재배치간격: 방중재의 배치간격을 입력합니다.
* 방충재형식: 방충재의 형식을 입력합니다.
* 계선주수량1: 계선주1의 수량을 입력합니다.
* 계선주수량2: 계선주2의 수량을 입력합니다.
* 계선주규격1: 계선주1의 규격을 입력합니다.
* 계선주규격2: 계선주2의 규역을 입력합니다.
* 계선주배치간격1: 계선주1의 배치간격을 입력합니다.
* 계선주배치간격2: 계선주2의 배치간격을 입력합니다.
* 계선주견인력1: 계선주1의 견인력을 입력합니다.
* 계선주견인력2: 계선주2의 견인력을 입력합니다.
* 주요취급화물: 주로 취급하는 화물을 입력합니다.
* 주요계류선박: 주로 계류한 선박을 입력합니다.
* 소재지: 시설의 위치를 입력합니다.
* 시작점위치: 시작점의 위치를 입력합니다.
* 종착점위치: 종착점의 위치를 입력합니다.
* 폭: 넓이를 입력합니다.
* 길이: 길이를 입력합니다.
* 포장종류: 포장종류를 입력합니다.
* 설계파고: 파의 골에서 마루까지의 높이를 입력합니다.
* 파랑주방향: 파랑주의 방향을 입력합니다.
* 토목시설물분류코드: 선택버튼을 클릭하여 분류코드를 입력합니다.
* 외축소파공경사비율: 외축소 파공 경사비율을 입력합니다.
* 외축소파공피복: 외축소 파공 피복을 입력합니다.
* 내축소파공경사비율: 내축소 파공 경사비율을 입력합니다.
## 내축소파공피복: 내축소 파공 피복을 입력합니다.

!! 특이사항
* 토목시설 일반에서는 GIS 자산 및 시설분류, 시설물 관리 그룹은 반드시 입력해야 한다.
* 토목시설 제원에서는 필요한 항목만 입력해도 무방하다.

!!! 토목시설 제원 관리(토목시설 첨부파일)
[이미지]

!! 업무 내용
* 토목시설 제원의 문서, 사진을 업로드, 다운로드, 삭제, 저장합니다.
* 업로드 한 사진을 미리보기 합니다.

!! 사용 설명
* 조회
## 토목시설 목록 화면에서 조회한 자료를 선택하고 토목시설 첨부파일 탭을 클릭합니다.
* 업로드
## 업로드버튼을 클릭합니다.
## 첨부파일 업로드화면에서 파일추가버튼을 클릭합니다.
## 업로드 할 파일을 선택하고 확인버튼을 클릭합니다.
## 토목시설 첨부파일 화면에 저장버튼을 클릭합니다.
* 다운로드
## 다운로드할 자료를 선택하고 다운로드버튼을 클릭합니다.
* 삭제
## 삭제할 자료를 선택하고 삭제버튼을 클릭합니다.

!! 항목 설명
* 파일구분: 문서인지 사진인지 입력합니다.
* 파일제목: 파일명이 아닌 파일의 이름을 입력합니다.

!! 특이사항
* 첨부파일은 문서, 사진, 기타로 구분된다. (사진일 경우 우측에서 프리뷰 가능)
