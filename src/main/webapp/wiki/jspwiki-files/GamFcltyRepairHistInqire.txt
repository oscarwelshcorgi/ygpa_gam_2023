!!! 시설물 하자보수이력 조회
[이미지]

!! 업무 내용
* 시설물관리자가 시설물 하자보수 이력을 조회합니다.

!! 사용 설명  
* 조회
## 시공업체명, 하자보수시설명, 하자보수공사시작일을 입력하여 조회버튼을 클릭합니다.  
  
!! 항목 설명
* 시공업체명: 하자보수 시공업체명을 입력합니다.
* 하자보수시설명: 조회할 하자보수시설명을 선택합니다.
* 하자보수공사시작일: 하자보수공사시작일의 범위를 입력합니다. 
  => 입력한 날짜 범위내의 모든 하자보수 내역을 조회합니다.

!! 특이사항
* 업무 구분을 정확히 확인한 후 사용한다.(건축시설, 토목시설, 기계시설, 전기시설, 정보통신시설)
* 시설물 관리 그룹이 없을 경우 시설물 관리 그룹 관리 화면에서 확인한다.
* 계약정보가 없을 경우 시설물 계약정보 관리 화면에서 확인한다.


!!! 시설물 하자보수이력 조회(하자보수내역 조회상세)
[이미지]

!! 업무 내용
* 하자보수이력목록을 선택했을 시 해당 하자보수 상세내역을 출력합니다. 

!! 사용 설명  
* 조회
## 시공업체명, 하자보수시설명, 하자보수공사시작일을 입력하여 조회버튼을 클릭합니다.  
## 조회버튼 클릭 시 하자보수내역 목록으로 Tab이 이동되며 조회조건에 해당하는 목록을  출력합니다.

!! 항목 설명
* 시행년도: 유지보수 시행년도를 출력합니다.
* 하자유무: 하자유무를 출력합니다.
* 시설물업무구분: 입력 시 시설물 구분값을 출력합니다. 
  => 전기시설물, 기계시설물, 토목시설물, 건축시설물, 정보통신시설물
* 하자보수유형: 하자보수유형을 출력합니다.
* 시설물관리그룹: 입력 시 하자보수 대상 시설물의 관리그룹을 출력합니다. 
* 계약번호: 해당시설물의 계약번호를 출력합니다.
* 하자보수순번: 하자보수순번을 출력합니다.
* 하자보수명: 하자보수명을 출력합니다.
* 하자보수업체명: 하자보수업체명을 출력합니다.
* 하자보수완료여부: 하자보수완료여부를 출력합니다.
* 하자보수금액: 하자보수금액을 출력합니다.
* 하자검사구분: 하자검사구분을 출력합니다.
=> 상반기, 하반기
* 하자발생일자: 하자발생일자를 출력합니다.
* 하자검사일자: 하자검사일자를 출력합니다.
* 하자보수시작일자: 하자보수시작일자를 출력합니다.
* 하자보수종료일자: 하자보수종료일자를 출력합니다.
* 하자보수내용: 하자보수내용을 출력합니다.
* 하자보수결과: 하자보수결과를 출력합니다.
* 비고: 기타 사항을 출력합니다.
