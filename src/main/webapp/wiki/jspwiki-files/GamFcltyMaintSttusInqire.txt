!!! 시설물 유지보수현황 조회
[이미지]

!! 업무 내용
* 시설물관리자가 시설물 유지보수 내역을 조회합니다.

!! 사용 설명  
* 조회
## 시설물관리그룹, 공사계약코드, 시설물업무구분, 유지보수공사명, 유지보수구분, 유지보수 공사시작일을 입력하여 조회버튼을 클릭합니다.  
  
!! 항목 설명
* 시설물관리그룹: 유지보수 대상 시설물의 관리그룹을 선택합니다.
* 공사계약: 해당 시설물의 유지보수에 해당하는 공사계약코드입니다. 
* 시설물업무구분: 시설물구분값을 선택합니다.
  => 전기시설물, 기계시설물, 토목시설물, 건축시설물, 정보통신시설물
* 유지보수구분: 유지보수 성격에 따른 구분내역입니다.
  => 개량, 보수, 보강, 변경-증설, 변경-구조변경, 기타
* 유지보수공사명: 신규 유지보수 입력 시 입력한 유지보수 공사명입니다.
  => 공사명의 전체 또는 일부를 입력하여 검색할 수 있습니다.
* 유지보수공사시작일: 신규 유지보수 입력 시 입력한 공사기간 중 첫 번째 항목입니다. 
  => 입력한 날짜 범위내의 모든 유지보수 내역을 조회합니다.

!! 특이사항
* 업무 구분을 정확히 확인한 후 사용한다.
(건축시설, 토목시설, 기계시설, 전기시설, 정보통신시설)
* 시설물 관리 그룹이 없을 경우 시설물 관리 그룹 관리 화면에서 확인한다.
* 계약정보가 없을 경우 시설물 계약정보 관리 화면에서 확인한다.

!!! 시설물 유지보수현황 조회(유지보수내역 상세)
[이미지]

!! 업무 내용
* 유지보수내역목록을 선택했을 시 해당 유지보수 상세내역을 출력합니다. 

!! 사용 설명  
* 조회
## 시설물관리그룹, 공사계약코드, 시설물업무구분, 유지보수공사명, 유지보수구분, 유지보수 공사시작일을 입력하여 조회버튼을 클릭합니다.
## 조회버튼 클릭 시 유지보수내역 목록으로 Tab이 이동되며 조회조건에 해당하는 목록을  출력합니다.

!! 항목 설명
* 시행년도: 유지보수 시행년도를 출력합니다.
* 시설물업무구분: 입력 시 시설물 구분값을 출력합니다. 
  => 전기시설물, 기계시설물, 토목시설물, 건축시설물, 정보통신시설물
* 유지보수구분: 유지보수 성격에 따른 구분내역입니다.
  => 개량, 보수, 보강, 변경-증설, 변경-구조변경, 기타
* 시설물관리그룹: 입력 시 유지보수 대상 시설물의 관리그룹을 출력합니다. 
* 유지보수순번: 유지보수순번을 출력합니다.
* 계약번호: 해당시설물의 계약번호를 출력합니다.
* 공사기간: 유지보수 공사기간을 출력합니다.
* 공사명: 유지보수 공사명을 출력합니다.
* 유지보수부위: 유지보수 부위를 출력합니다.
* 예산: 집행될 유지보수 예산금액을 출력합니다.
* 설계자: 유지보수 설계담당자를 출력합니다.
* 시공자: 유지보수 시공자를 출력합니다.
* 공사금액: 실제 유지보수 공사금액을 출력합니다.
* 책임기술자: 해당 유지보수의 책임기술자를 출력합니다.
* 공사감독자: 해당 유지보수의 공사감독자를 출력합니다.
* 유지보수내용: 실제 유지보수된 상세 내용을 출력합니다.
* 비고: 기타 사항을 출력합니다.
