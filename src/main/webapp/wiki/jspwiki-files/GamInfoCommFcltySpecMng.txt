!!! 정보통신시설 제원 관리
[이미지]

!! 업무 내용
* 정보통신시설 제원을 등록 관리합니다.

!! 사용 설명  
* 조회
## 항구분, 시설물관리그룹, 시설분류, 시설명, 소재지를 입력하고 조회버튼을 클릭합니다.
* 입력
## 시설추가버튼을 클릭하여 정보통신시설 제원 화면에 해당하는 자료를 입력하고 저장버튼  을 클릭합니다.
* 수정
## 조회조건을 입력하여 수정할 자료를 조회합니다. 
## 정보통신시설 목록 화면에서 해당 자료를 더블클릭합니다.
## 정보통신시설 제원 화면에서 해당 자료를 수정하고 저장버튼을 클릭합니다.
* 삭제
## 조회조건을 입력하여 삭제할 자료를 조회합니다. 
## 정보통신시설 목록 화면에서 삭제할 자료를 선택하고 삭제버튼을 클릭합니다.  
* 맵조회
## 조회조건을 입력하여 맵조회할 자료를 조회합니다. 
## 맵조회할 자료를 선택하고 맵조회버튼을 클릭합니다. 

!! 항목 설명
* 항구분: 관리기관입니다
* 시설물관리그룹: 시설물을 그룹으로 관리 합니다.
* 시설분류: 시설물을 관리기관으로 분류합니다.
* 시설명: 시설물 이름입니다.
* 소재지: 시설물 위치입니다.

!! 특이사항
* 정보통신시설 제원만 조회된다.(만약 조회결과에 해당 시설이 없을 경우 추가로 입력한다.)
* 맵조회는 그리드에서 해당 시설을 선택한 후 정상적으로 조회가 가능하다.

!!! 정보통신시설 제원 관리(정보통신시설 제원)
[이미지]

!! 업무 내용
* 정보통신시설 제원의 자료를 입력, 수정, 출력합니다.
* 정보통신시설 제원의 위치를 등록 조회 합니다.

!! 사용 설명
* 조회
## 정보통신시설 목록 화면에서 조회한 자료를 더블클릭합니다.
* 입력
## 정보통신시설 목록 화면에서 시설추가버튼을 클릭하여 정보통신시설 제원 화면에
해당하는 자료를 입력하고 저장버튼을 클릭합니다.
* 수정
## 조회조건을 입력하여 수정할 자료를 조회합니다. 
## 정보통신시설 목록 화면에서 조회한 자료를 더블클릭합니다.
## 정보통신시설 제원 화면에서 해당 자료를 수정하고 저장버튼을 클릭합니다.
* 위치등록
## 위치등록버튼을 클릭하여 위치를 등록합니다.
* 위치조회
## 위치조회버튼을 클릭하여 위치를 조회합니다.

!! 항목 설명
* 입력항목 설명
* 항코드: 관리기관입니다
* GIS자산코드: 선택버튼을 클릭하여 GIS자산코드를 입력합니다.
* GIS자산명: GIS자산코드를 입력하면 자동 부여 됩니다.
* 지번: GIS자산코드를 입력하면 자동 부여 됩니다.
* 소재지: GIS자산코드를 입력하면 자동 부여 됩니다.
* 시설코드: 시설분류를 입력하면 자동 부여 됩니다.
* 시설분류: 시설물을 관리기관을 표시합니다.
* 시설명: 정보통신시설 이름을 표시합니다.
* 시설물관리그룹: 시설물을 그룹으로 관리 합니다.
* 시설규격: 시설의 품질, 모양, 크기, 성능의 일정한 표준을 입력합니다.
* 설치일자: 시설의 설치 날짜를 입력합니다.
* 변경일자: 시설의 변경 날짜를 입력합니다.
* 품목명: 제품의 이름을 입력합니다.
* 모델: 제품의 고유 코드를 입력합니다.
* 제조사: 제품을 만든 회사의 이름을 입력합니다.
* 수량: 제품의 개수를 입력합니다.
* 규격: 제품의 품질, 모양, 크기, 성능의 일정한 표준을 입력합니다.
* 세부내역: 제품의 세부 내역을 입력합니다.
* 설치구분: 제품의 설치 구분을 입력합니다.
* 설치번호: 제품의 설치 번호를 입력합니다.
* 기능: 제품의 기능을 입력합니다.
* 설치높이: 제품의 설치 높이를 입력합니다.
* 설치규격: 시설의 품질, 모양, 크기, 성능의 일정한 표준을 입력합니다.
* 제어방식: 제품의 제어 방식을 입력합니다.
* LAMP형식: 제품의 램프 형식을 입력합니다.
* 시설물분류코드: 선택버튼을 클릭하여 분류코드를 입력합니다.
* 건축시설물관리번호: 선택버튼을 클릭하여 관리번호를 입력합니다.
* 소재지: 시설의 위치를 입력합니다.
* 비고: 기타 사항을 입력합니다.

!! 특이사항
* 정보통신시설 일반에서는 GIS 자산 및 시설분류, 시설물 관리 그룹은 반드시 입력해야 한다.
* 정보통신시설 제원에서는 필요한 항목만 입력해도 무방하다.

!!! 정보통신시설 제원 관리(정보통신시설 첨부파일)
[이미지]

!! 업무 내용
* 정보통신시설 제원의 문서, 사진을 업로드, 다운로드, 삭제, 저장합니다.
* 업로드 한 사진을 미리보기 합니다.

!! 사용 설명
* 조회
## 정보통신시설 목록 화면에서 조회한 자료를 선택하고 정보통신시설 첨부파일 탭을  클릭합니다.
* 업로드
## 업로드버튼을 클릭합니다.
## 첨부파일 업로드화면에서 파일추가버튼을 클릭합니다.
## 업로드 할 파일을 선택하고 확인버튼을 클릭합니다.
## 정보통신시설 첨부파일 화면에 저장버튼을 클릭합니다.
* 다운로드
## 다운로드할 자료를 선택하고 다운로드버튼을 클릭합니다.
* 삭제
## 삭제할 자료를 선택하고 삭제버튼을 클릭합니다.

!! 항목 설명
* 파일구분: 문서인지 사진인지 입력합니다.
* 파일제목: 파일명이 아닌 파일의 이름을 입력합니다.

!! 특이사항
* 첨부파일은 문서, 사진, 기타로 구분된다. (사진일 경우 우측에서 프리뷰 가능)
